# Claude Code Project Instructions

## Commit Guidelines

When committing changes to this repository:

1. **Author**: Always commit using Claude as the commit author
2. **Commit Message Format**:
   - Use a clear, descriptive commit title that summarizes the change
   - Include a robust description in the commit body that explains:
     - What was changed
     - Why the change was made
     - Any relevant context or implications
   - Follow conventional commit style when appropriate

Example commit format:
```
feat: Add user authentication module

This commit introduces a new authentication system that supports:
- Email/password login
- OAuth integration with Google and GitHub
- Session management with secure cookie handling

The implementation follows security best practices including
password hashing with bcrypt and rate limiting on login attempts.
```
